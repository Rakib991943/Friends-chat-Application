# 📱 Chat Application

## 1. Application Name
Chat Application

---

## 2. About This Application
This is a chat application web platform.  
It can be used for social communication like WhatsApp, Messenger, and Facebook.

Users can:
- Send text messages
- Make video calls
- Make audio calls
- Interact with friends in a social media style interface

This application is designed for everyone and includes multiple interactive features with modern UI/UX.

---

## 3. Technologies Used
- Next.js
- JavaScript
- Tailwind CSS
- DaisyUI
- React Icons

---

## 4. Key Features

### 🏠 Home Page
- Displays all available friends/data
- Shows dynamic content from JSON/API
- Click on a specific friend to view detailed information page

---

### ⏱ Timeline Page
- Shows selected friends activity
- Displays interactions like:
  - Text messages
  - Audio calls
  - Video calls
- Data is dynamically updated based on user actions

---

### 📊 Status Page
- Displays dynamic user status data
- Visualized using charts (Pie Chart)
- Shows activity summary and statistics

---

### 📄 Details Page
- Shows specific user or friend information
- Accessible from Home page selection

---

## 5. UI Libraries
- DaisyUI for UI components
- React Icons for modern icons

---
